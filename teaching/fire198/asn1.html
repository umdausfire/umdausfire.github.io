ASN1

Setup Your Computer To Bash It!

Deadline


Feb 01, 2022 at 11:59:59 PM (Individual Submissions).



What you need to do?


This assignment has two tasks. Your first task is to setup your computer to be able to run the required softwares for the course. In the second task, you'll learn about a few basic functions to work with the Terminal/Shell.

Step 1: Setup Your Computer


Follow the instructions for setting up your computer based on the Operating System you have.

Windows

Install Windows Subsystem for Linux (WSL1 or WSL2) and then install Ubuntu 18.04 or 20.04. Now, install XMing so that you can forward your display. Inside the Ubuntu environment, install the following packages Python >=3.8, python-opencv, av and the following Python-pip packages numpy, scipy.

Linux (My Preferred OS!)

Install the following packages Python >=3.8, python-opencv, av and the following Python-pip packages numpy, scipy.

MacOS (Intel/AMD)

Install XCode and then the following packages Python >=3.8. Now, install the following Python-pip packages av, opencv-python, numpy, scipy.

.
MacOS (M1 or M1 Pro or M1 Max)

Install the UTM Virtual Machine with Ubuntu 20.04 from here. Now follow the instrctions on Linux inside your Virtual Machine. Alternatively (if you did the previous UTM method, do not do any of the further steps), you can install everything in an Anaconda environment as given here or here. If you are super ambitious (not recommended method), you can natively install XCode and then the following packages Python >=3.8. Now, install the following Python-pip packages av, numpy, scipy. To install OpenCV package, you can follow the instructions from here.

Alternatively (for Windows and Mac), you can also utilize a Virtual Machine to run Ubuntu. For Windows and Intel/AMD Mac, I recommend using VirtualBox and for M1 Mac I recommend using UTM (Amazing blog post on this can be found here) or Parallels.

Step 2: Learn Basic Terminal/Bash functions


Find answers to the following questions using the Terminal/Bash (for WSL's terminal (Ubuntu) if using Windows, native terminal if using Linux/MacOS), you'll need to make a document to fill in the answers. More details about the submission is given in the next section.
  • How do you run a command with superuser access?
  • How do you list (a) only non-hidden files?, (b) non-hidden and hidden files?
  • When you list files, how do you distinguish between files and folders?
  • How do you (a) Change directory to any location?, (b) Change directory to home location?, (c) Change directory to one level up/previous?
  • How do you (a) Make a folder?, (b) Make a folder with sub-folders?
  • How do you (a) Remove a file?, (b) Remove all files in a folder? (BE SUPER CAREFUL WHEN RUNNING THESE COMMANDS! IT CAN WIPE YOUR ENTIRE HARD-DRIVE!)
  • How do you (a) Move a file from one location to another?, (b) Move a folder from one location to another?
  • How do you (a) Copy a file from one location to another?, (b) Copy a folder from one location to another?
  • How do you display the contents of a file on the terminal?
  • How do you know the current working directory path?
  • How do you clear the contents of the terminal?


What you need to submit?


A document (feel free to use Word or Google Docs or any other software for this) converted to PDF with the following answers:
  • One or multiple screenshots with the outputs of the following commands in Python environment after importing cv2, numpy, scipy, av.
    • cv2.__version__
    • numpy.__version__
    • scipy.__version__
    • av.__version__
    If any of the above commands throw an error, you have not installed the respective package correctly.
  • Answer all the questions in Step 2 in the same document.
  • Positive or Negative feedback about this assignment if you have any.
  • Challenges you faced in solving this assignment along with lessons learned.
