ASN8

FIRE Summit Deliverables!

Deadline


Presentation in-class on Oct 26, 2021. Presentation file due at 11:59:59 PM on same day.



What you need to do?


You need to make a professional presentation (any software of your choice but has to be professional, I recommend not to use Prezi) for a 5 min talk given by one member of the team. Keep the slides with minimum text, more media content is highly encouraged with a good set of animations so as to not overload the observer. You can treat the class slides as an idea of text to media ratio. Media can include videos and images. Use softwares like Photoshop or draw.io or Corel Draw to make high quality images. Use a good camera (a good phone camera with protrait mode if needed) to capture compelling photos of your project to be used in slides. This talk is NOT for an expert in the field but for the layperson, Do not make it overly technical but technical enough that it shows the complexity to the expert observer. You can use the FIRE Summit template provided here as a starting point, but feel free to make your own slides. I will grade you not only presentation quality but also attention to detail and how creatively you present your approach.

What you need to submit?


Your presentation file in the format it was made in (for example this would be .pptx if you used Microsoft Powerpoint) on ELMS along with a live presentation session during the deadline time indicated above.
